# react-form-schema-json

## Schema
```
[
  {
    "type"        : "inputText",
    "_id"         : "text",
    "label"       : "Text",
    "placeholder" : "Enter Text here"
  },
  {
    "type"        : "inputPassword",
    "_id"         : "password",
    "label"       : "Password",
    "placeholder" : "Enter password here"
  },
  {
    "type"        : "inputOption",
    "_id"         : "option",
    "label"       : "Option",
    "options"     : [
      {
        "text":"Yes",
        "value":true
      },
      {
        "text":"No",
        "value":false
      }
    ]
  }
]
```


Running :
> $ npm install <br>
> $ npm start

> open your browser.

Thanks to create-react-app boilerplate.
